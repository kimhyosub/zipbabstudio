🍚 집밥연구소관리자 모드

이 데이터는 브라우저 저장소(localStorage)에 저장됩니다. 기기·브라우저가 바뀌면 유지되지 않을 수 있습니다.

이 화면은 워드프레스 느낌의 정적 CMS-lite 관리자 UI 데모입니다. 실제 데이터베이스·인증·권한 관리 시스템이 없으며, 여기서 저장한 내용은 이 브라우저의 localStorage에만 남고 실제 배포된 페이지 파일에는 반영되지 않습니다. 실제로 사이트에 반영하려면 JSON export 후 그 내용을 참고해 data/posts.js, data/columns.js 등과 해당 HTML 페이지를 직접 수정해야 합니다. 추후 Supabase, Firebase, Git 기반 CMS 등과 연동하면 실시간 반영도 가능합니다.
